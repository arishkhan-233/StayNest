# StayNest

StayNest is a full-stack property rental web application where users can explore, create, and manage property listings with secure authentication and authorization.

## Features

- Secure User Authentication
- Authorization & Ownership-Based Access Control
- Users can create their own property listings.
- Users can edit and delete only their own listings.
- Users can add reviews to any listing.
- Users can edit and delete only their own reviews.
- Unauthorized users cannot edit or delete other users listings or reviews.
- Image upload with Cloudinary
- Flash messages and server-side validation
- Responsive and user-friendly interface

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap
- Passport.js
- Cloudinary
- Joi

## Installation

```bash
git clone https://github.com/yourusername/StayNest.git
cd StayNest
npm install
npm start
```
